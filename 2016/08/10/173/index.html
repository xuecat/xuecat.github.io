std::enable_if

源代码：

template<class _Ty>
struct enable_if<true, _Ty>
{    // type is _Ty for _Test
typedef _Ty type;
};

其实它和static_assert一样，都是为了编译期间检查。常用使用在函数前，表面上给个函数返回值，但之前要进行判断。
